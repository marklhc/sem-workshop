Full SEM

From CFA to Full SEM

What is Full SEM?

SEM = CFA (measurement) + Path analysis (structural)

  • Measurement model: how latent variables are measured by indicators
  • Structural model: directional relations among latent variables

Unlike observed-score path models, SEM explicitly accounts for measurement error.

An Example of Full SEM

Why Use Full SEM?

  • We can test mediation or direct effects among latent constructs
  • Path coefficients are less attenuated by measurement error
  • Model fit evaluates whether the full covariance structure is plausible

Note

SEM is especially useful when your substantive hypotheses are causal but your key variables are latent.

Example: SOP, NGSE, and PASS

Example Data

  • Data source: Williams & Edwards (2021, conceptual replication of Seo, 2008)
  • Sample size: \(N = 575\)
  • Constructs:
    • SOP: Self-Oriented Perfectionism (5 items)
    • NGSE: General Self-Efficacy (8 items)
    • PASS: Academic Procrastination (4 parcels: assignments, exams, reading, general)

Hypothesized Mediation Model

Figure 1

\[ \begin{aligned} \text{NGSE} &= a\cdot\text{SOP} + \zeta_1 \\ \text{PASS} &= b\cdot\text{NGSE} + c'\cdot\text{SOP} + \zeta_2 \end{aligned} \]

Two-Step SEM Workflow

  1. Step 1: Full Measurement model (CFA)
    • Verify indicators measure intended factors
    • Check global fit and loading pattern
  2. Step 2: Structural model (SR/SEM)
    • Impose directional paths among latent factors
    • Evaluate direct, indirect, and total effects

Note

If Step 1 is poor, Step 2 interpretations are not trustworthy.

Step 2: Structural Model

SR Model Specification

model_sr_we <- "
  # Measurement model
  SOP  =~ SOP_1 + SOP_2 + SOP_3 + SOP_4 + SOP_5
  NGSE =~ NGSE_1 + NGSE_2 + NGSE_3 + NGSE_4 +
          NGSE_5 + NGSE_6 + NGSE_7 + NGSE_8
  PASS =~ Pasgn + Pexam + Pread + Pgen

  # Structural model (mediation)
  NGSE ~ a * SOP
  PASS ~ b * NGSE + c * SOP

  # Defined parameters
  indirect := a * b
  total    := c + a * b
"
fit_sr_we <- sem(
    model_sr_we,
    data = we_data,
    std.lv = TRUE,
    estimator = "MLR",
    missing = "FIML"
)
summary(fit_sr_we, fit.measures = TRUE, standardized = TRUE, rsquare = TRUE)

Interpretation of Effects

  • Indirect effect: \(a \times b\) (SOP \(\rightarrow\) NGSE \(\rightarrow\) PASS)
  • Direct effect: \(c'\) (SOP \(\rightarrow\) PASS, controlling for NGSE)
  • Total effect: \(c' + a\times b\)

Note

If the 95% CI for the indirect effect excludes 0, the mediation effect is statistically supported.

Brief Causal Caution

  • A fitted SEM does not prove causal direction
  • Causal interpretation depends on assumptions (no key omitted confounders)
  • Equivalent/reversed models may show similar global fit

Warning

Use theory and design (not fit indices alone) to justify directional claims.

Summary

  • Structural regression combines CFA and path analysis in one model
  • Use the two-step strategy: validate measurement first, then test structure
  • Indirect effects are straightforward in lavaan using :=
  • Good fit supports plausibility, but does not establish causality by itself
